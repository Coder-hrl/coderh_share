##### map 函数

**第一个参数是回调函数**,第二个参数是给**回调函数绑定 this**

回调函数也有三个参数,**第一个参数是这一次执行时的元素**,**第二个是索引值**,**第三个完整的数组对象**

```js
this.state.movies.map(item => {
	return <li>{item}</li>;
});
```
