### 严格模式

> 1. 只会检测包裹的后代元素,**他会识别一些不安全的生命周期,对于一些不安全的生命周期会进行警报.**
> 2. 使用过期的API方法,如使用字符串的ref
> 3. 会不会有意外产生的副作用的回调函数
> 4. 检测过时的contextAPI方法

```js
<React.StrictMode><App/> </React.StrictMode>
```

他只是让代码变得更加健壮,`触发严格模式的检查和警告`

`在开发模式中使用,并不在生产模式下使用`
