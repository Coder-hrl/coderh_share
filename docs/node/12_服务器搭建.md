### 运行, pm2 运行环境

> Pm2 的运行环境是自动运行的,让在退出命令行的输入后,仍然可以运行 Node 进程来进行
> 是让 node 开发的服务程序运行的不可缺少的一物

```shell
pm2 start ./src/main.js --name coderh
运行环境

pm2 list

pm2 stop all
```
